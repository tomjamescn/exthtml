Setup

Install Xel

Run from console:

npm install xel

Link Xel

Add to the <head>:

<link rel="import" href="node_modules/xel/xel.min.html">

Link Xel theme

Add to the <head> one of the following:

Material: 

<link rel="stylesheet" href="node_modules/xel/stylesheets/material.theme.css">

macOS: 

<link rel="stylesheet" href="node_modules/xel/stylesheets/macos.theme.css">

Material theme expects Roboto and Roboto Mono fonts to be installed on the system or embedded by your app. Otherwise, it will try to fetch them from Google Fonts servers and if this fails as well, it will fallback to the default system font.

MacOS theme expects San Fransisco fonts to be installed on the system. Otherwise, it will fallback to simillar more genereric families such as Helvetica Neue, Lucida Grande or Arial.
